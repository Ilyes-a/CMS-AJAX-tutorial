Présentation du projet

Ce projet permet de connecter un CMS (WordPress, Joomla, WebsiteBaker) à un Système d’Information indépendant développé en PHP. Le système gère l’authentification, les réservations et les profils utilisateurs via une API REST sécurisée.

Choisissez votre CMS

Pour accéder aux instructions d’intégration, sélectionnez le CMS souhaité dans le menu ci-dessus.
