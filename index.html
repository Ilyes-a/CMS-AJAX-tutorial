Présentation du Projet

Ce projet a pour objectif de connecter un Système d’Information (SI) indépendant, qui gère l’authentification, la réservation et la gestion des profils utilisateurs via une API PHP et AJAX (SPA), à différents CMS (WordPress, Joomla, WebSiteBaker) sans modifier leur code natif.

Le SI se charge du contenu dynamique (pages HTML, CSS et JavaScript) tandis que le CMS continue de gérer le contenu statique (articles, menus, blog).

Choisissez votre CMS dans le menu pour accéder aux instructions détaillées.
