Fonctionnement Général

  • API : Gère les appels AJAX (authentification, inscription, réservation, etc.).
  • Vues SPA : Les pages HTML sont chargées dynamiquement via fetch().
  • Router : Le fichier router.js adapte les chemins en fonction du CMS appelant.
  • Redirect : redirect.php permet de simuler des routes classiques (utile pour WebSiteBaker).
