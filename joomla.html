Guide d'intégration de votre SI dans Joomla

Ce guide vous explique comment intégrer votre système d'information (SI) dans Joomla en réalisant les opérations suivantes :

  • Création des articles pour chaque page (accueil, login, register, mon-compte, réserver).
  • Configuration du menu pour obtenir des URLs propres.
  • Création d'un module personnalisé global pour la navigation dynamique.
  • Utilisation de CodeMirror pour éviter le filtrage du code JavaScript.

1. Création des articles

Pour chaque page de votre SI (Accueil, Login, Register, Mon Compte, Réserver), créez un article dans Joomla :

  1. Connectez-vous à l'administration Joomla et accédez à Contenu > Articles.
  2. Cliquez sur Nouveau pour créer un article.
  3. Collez le contenu HTML généré par votre SI dans l'éditeur.
  4. Assurez-vous que l'éditeur est en mode CodeMirror.
  5. Enregistrez l'article avec un alias correspondant (ex : login).

2. Configuration du menu

Pour obtenir des URLs propres (SEF), procédez ainsi :

  1. Accédez à Système > Configuration globale.
  2. Dans l'onglet Site, activez :
    • Search Engine Friendly URLs
    • Use URL Rewriting
  3. Renommez le fichier htaccess.txt en .htaccess.
  4. Créez des éléments de menu dans Menus > Main Menu avec les bons alias (login, register...)

3. Module personnalisé global pour la navigation

Créez un module de type HTML personnalisé :

  1. Accédez à Extensions > Modules.
  2. Créez un nouveau module HTML personnalisé.
  3. Passez en mode CodeMirror et désactivez Préparer le contenu.
  4. Collez ce code : 
    <div id="global-menu">
  <ul>
    <li id="menu-accueil"><a href="/">Accueil</a></li>
    <li id="menu-login"><a href="/login">Login</a></li>
    <li id="menu-register"><a href="/register">Register</a></li>
    <li id="menu-mon-compte"><a href="/mon-compte">Mon Compte</a></li>
    <li id="menu-reserver"><a href="/reserver">Réserver</a></li>
  </ul>
</div>

<script>
  document.addEventListener("DOMContentLoaded", () => {
    const token = localStorage.getItem("jwt");

    if (token) {
      document.getElementById("menu-login").style.display = "none";
      document.getElementById("menu-register").style.display = "none";
    } else {
      document.getElementById("menu-mon-compte").style.display = "none";
      document.getElementById("menu-reserver").style.display = "none";
    }
  });
</script>

4. Configuration de CodeMirror

  1. Accédez à Système > Configuration globale.
  2. Sous Editeur par défaut, choisissez CodeMirror.

Conclusion

Vous pouvez maintenant intégrer efficacement votre SI dans Joomla avec :

  • Des articles en mode code (non filtrés)
  • Des URLs propres via les menus et SEF
  • Un module de menu dynamique en fonction de l'état de connexion
